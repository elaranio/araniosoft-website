# i18n Design — araniosoft.com
Date: 2026-02-27

## Goal
Add English and French translations to araniosoft.com using static subdirectories and browser-language auto-detection.

## Approach: Static subdirectories + auto-redirect

Chosen over JS content-swapping (bad SEO) and build-step templates (breaks no-build architecture).

## File Structure

```
/index.html                  → ES default (+ redirect snippet + hreflang)
/zeroBudget/index.html       → ES default (+ redirect snippet + hreflang)

/en/index.html               → English (new)
/en/zeroBudget/index.html    → English (new)

/fr/index.html               → French (new)
/fr/zeroBudget/index.html    → French (new)
```

## Auto-redirect

Only on ES root pages. Reads `navigator.language`, redirects to `/en/` or `/fr/` if matched. EN/FR pages do NOT redirect.

```js
(function(){
  var l = (navigator.language||'').toLowerCase().split('-')[0];
  if (l==='en') location.replace('/en/');
  else if (l==='fr') location.replace('/fr/');
})();
```

## SEO

Every page gets:
- `<link rel="alternate" hreflang="es" href="https://araniosoft.com/">`
- `<link rel="alternate" hreflang="en" href="https://araniosoft.com/en/">`
- `<link rel="alternate" hreflang="fr" href="https://araniosoft.com/fr/">`
- `<link rel="alternate" hreflang="x-default" href="https://araniosoft.com/">`
- `<link rel="canonical">` pointing to its own URL
- `og:locale`: `es_MX` / `en_US` / `fr_FR`

Sitemap.xml updated to include all 6 URLs.

## Content Adaptations (EN/FR vs ES)

| Element | ES | EN | FR |
|---|---|---|---|
| Badge país | 🇲🇽 Hecho en México | *(removed)* | *(removed)* |
| Bank import feature | BBVA, Banorte, HSBC, Santander... | CSV/Excel from any bank | CSV/Excel de n'importe quelle banque |
| Siri example | "agrega gasto de 500 en walmart" | "add $50 expense at Walmart" | "ajouter une dépense de 50€ chez supermarché" |
| Notify email subject | Spanish subject | English subject | French subject |
| Privacy/Terms links | ES pages | Link to ES pages (not translated in this phase) | Link to ES pages |

## Out of Scope

- `privacy.html` and `terms.html` translations (lower traffic, legal complexity)
- Visible language switcher (user chose auto-detect only)
- Screenshots localization (app UI is already in English in simulator)
